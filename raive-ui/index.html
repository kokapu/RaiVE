How to use RaiVE

Basic Controls

RaiVE is a tool to generate simple electronic music by converting your musical ideas into playable code. Type a prompt in the input box and press Enter to generate music. See prompt examples below for ideas!


To build your musical piece, you can either directly edit the code or type another prompt to append to the existing code.

About RaiVE

What is this?

RaiVE is a tool meant to make the creation of electronic music more accessible. We had a lot of fun making it, and we hope you have fun using it! We are constantly trying to make RaiVE better, so please reach out to us if you have any suggestions!
